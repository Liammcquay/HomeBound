HomeBound Project

HomeBound is a charity project aimed to help support homeless shelters and orphanages within Thailand. We as a community would like to encourage all of you to help donate unused clothing as well as products to those in need. Money is also a useful component in helping them enter into a better life.

We also hope that this project will be able to give funds for their shelters in order for them to be able to live in a better environment. This will not only help those in need but will also help up create a better society.
